Mengapa ferzui?

Framework UI yang dirancang untuk developer modern dengan fitur-fitur terdepan dan performa tinggi.

🎨

Modern Design

Desain yang lebih modern dari Bootstrap dengan glassmorphism, gradients, dan animasi yang smooth.

🌙

Dark Mode

Support dark mode otomatis dengan prefers-color-scheme dan toggle manual untuk pengalaman terbaik.

📱

Mobile First

Responsive design yang mobile-first dengan breakpoints yang optimal untuk semua perangkat.

⚡

High Performance

Optimized untuk performa dengan lazy loading, code splitting, dan bundle size yang minimal.

♿

Accessible

WCAG AA compliant dengan keyboard navigation, screen reader support, dan high contrast mode.

🔧

Customizable

Mudah dikustomisasi dengan CSS variables, SCSS mixins, dan theme editor yang interaktif.
