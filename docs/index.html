Mengapa ferzui?

Framework UI yang dirancang untuk masa depan web development

🎨

Design System Modern

Fluid typography, spacing yang konsisten, dan shadow yang indah. Dibangun dengan prinsip desain modern yang memukau.

♿

Aksesibilitas Lengkap

WCAG AA compliant dengan dukungan ARIA lengkap, keyboard navigation, dan focus management yang sempurna.

🌙

Multi-Theme Support

Light, dark, dan high-contrast mode dengan CSS custom properties yang mudah dikustomisasi.

📱

Responsive Design

Mobile-first approach dengan breakpoint utilities yang komprehensif untuk semua ukuran layar.

🚀

Performance Optimized

Tree-shakeable CSS dan JavaScript ringan tanpa dependensi eksternal untuk performa maksimal.

🔧

Framework Agnostic

Bekerja dengan React, Vue, Angular, atau vanilla HTML. Tidak terikat dengan framework tertentu.

Quick Start

Mulai menggunakan ferzui dalam hitungan menit

1. Install via CDN

<!-- CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/ferzui@0.2.0/dist/ferzui.min.css"> <!-- JavaScript --> <script src="https://cdn.jsdelivr.net/npm/ferzui@0.2.0/dist/ferzui.min.js"></script>

2. Install via NPM

npm install ferzui

3. Basic HTML Template

<!DOCTYPE html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My App</title> <link rel="stylesheet" href="ferzui.css"> </head> <body> <div class="container"> <h1>Hello ferzui!</h1> <button class="btn btn-primary">Click me</button> </div> <script src="ferzui.js"></script> </body> </html>
